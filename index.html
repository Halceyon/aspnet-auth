asp-net auth

A standalone lightweight javascript library for authenticating with ASP.NET OWIN site.

Features

    ✓ Authenticate to your existing ASP.NET page from a standalone HTML 5 app
    ✓ Authenticate to ASP.NET from mobile devices
    ✓ Best practice security implementation
    ✓ Generate a static site for your project via Easystatic
    ✓ Cross-platform with minimum dependencies
